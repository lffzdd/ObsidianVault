# 满足结合律
![[Pasted image 20241103224525.png]]
![[Pasted image 20241103224549.png]]
加法结合律: $a+b+c=(a+b)+c=a+(b+c)$
减法: $a-b-c=(a-b)-c\ne a-(b-c)$
为了使一对互逆元素的成立没有争议,不受到运算顺序的影响 (注意: 是运算的顺序, 而不是元素排列的顺序),那就要求我们所设定的乘法必须要满足结合律
