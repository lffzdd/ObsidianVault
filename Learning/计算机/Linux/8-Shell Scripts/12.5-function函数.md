```shell
function fname() {
	程序段
}
# 调用
fname()
```
function 拥有内建变量～他的内建变量与 shell script 很类似，函数名称代表示 $0 ，而后续接的变量也是以 $1, $2… 来取代的～ 这里很容易搞错喔～因为『 function fname() { 程序段 } 』内的 $0, $1… 等等与 shell script 的 $0 是不同的。
```shell
  1 #!/bin/bash                                                                                 
  2
  3 function func(){
  4     echo $1
  5 }
  6
  7 echo "func:$(func)"
  8 echo "func 1:$(func 1)"
```
