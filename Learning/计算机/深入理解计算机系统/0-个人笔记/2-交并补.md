# 看看异或
和 0 异或得到自己, 相当于元变换 e, 和 1 异或得到别人, 这个群只有两个元素,
所以例如
$$
1\oplus1\oplus0\oplus1\oplus\cdots\oplus\cdots
$$
其中的 $\oplus0$ 其实都可以去掉, 而每两个 $\oplus1$ 都可以抵消, $0$ 和 $1$ 的个数决定了最后的结果, 而顺序并没有影响, 所以满足交换律, 既然顺序不影响, 同时满足结合律
