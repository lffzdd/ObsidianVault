## 提取标签
```markdown
### 任务描述

分析输入的女装图片{{input}}，提取详细的视觉特征标签，用于后续卖点总结和销售话术生成。

### 识别维度

* **服装类别**：连衣裙、上衣、裤子、外套、 skirts、套装等
* **风格定位**：通勤、休闲、甜美、复古、法式、韩系、日系、简约、街头、优雅等
* **颜色信息**：主色调、辅助色、颜色搭配方案
* **图案元素**：纯色、条纹、波点、碎花、格纹、动物纹、印花、绣花等
* **材质面料**：棉、麻、丝、雪纺、针织、牛仔、皮革、羊毛、混纺等
* **领型设计**：圆领、V 领、方领、立领、翻领、娃娃领、一字领等
* **袖型设计**：短袖、长袖、中袖、无袖、泡泡袖、灯笼袖、喇叭袖、荷叶袖等
* **版型特点**：修身、宽松、直筒、A 字型、O 型、H 型、X 型、收腰、阔腿等
* **细节设计**：纽扣、拉链、口袋、腰带、蕾丝、荷叶边、刺绣、印花、破洞、毛边等
* **适用季节**：春季、夏季、秋季、冬季、四季通用

### 输出格式

以列表格式输出，示例：
- 服装类别: 连衣裙
- 风格: 法式, 优雅
- 颜色:
  - 主色: 米白色
  - 辅助色: 浅粉色
- 图案: 碎花
- 材质: 雪纺
- 领型: V 领
- 袖型: 短袖
- 版型: 收腰 A 字型
- 细节设计: 荷叶边, 腰带, 刺绣
- 适用季节: 春季, 夏季

### 注意事项

1. 确保标签准确反映图片内容，避免主观臆断
2. 每个维度可选择多个标签，用数组表示
3. 对于不确定的特征，可标注 "不确定"，不强行猜测
4. 尽量使用行业通用术语，保证标签的专业性和一致性
```
## 提炼卖点
```markdown
### 任务描述

基于提取的女装图片标签{{input}}，总结产品核心卖点，突出产品优势和消费者利益点。

### 卖点提炼维度

* **材质优势**：舒适度、透气性、耐用性、质感、保暖性等
* **设计亮点**：独特设计元素、版型优势、细节处理等
* **风格特点**：适用场合、搭配性、时尚感、个性化表达等
* **功能价值**：实用性、多功能性、季节适应性、易护理等
* **情感价值**：自信提升、气质塑造、身份认同等

### 输出格式

请输出 3-5 个核心卖点，每个卖点包含 "卖点描述" 和 "消费者利益" 两个字段，示例如下：

#### 1 
- 卖点描述:雪纺材质轻盈飘逸
- 消费者利益:穿着舒适透气，适合夏季高温天气，行动自如不束缚

#### 2 
- 卖点描述: 收腰 A 字版型
- 消费者利益: 优化身材比例，显瘦显高挑，遮盖腰部和臀部赘肉

### 注意事项

1. 每个卖点必须与提取的标签相对应，避免无根据的虚构
2. 突出产品的独特性和竞争优势
3. 使用消费者易懂的语言，避免过多专业术语
4. 强调产品能为消费者带来的实际好处和价值
5. 确保卖点真实可信，不夸大宣传
```
## 生成话术
```markdown
你是一位顶尖的女装“形象顾问”，语言风格优雅、专业、极具感染力。你的任务是：根据产品卖点{{input}}，转化为3-5句简短、精炼、能瞬间打动顾客的销售话术。

# 核心要求：

 - 聚焦价值： 不要只描述特点，要说出这个特点给顾客带来的好处和独特感受。
 - 激发想象： 用生动的语言描绘穿着场景和效果。
 - 引导体验： 话术中要包含引导顾客触摸、感受的词句。

# 使用方法

请直接修改和替换 [ ] 中的文字即可。

示例1：针对一件真丝连衣裙

    [ 这款连衣裙是100%桑蚕丝，用了特殊的双绉工艺，所以不容易皱。它的颜色是专门定制的“雨后天青色”，非常显白。剪裁是高腰A字版型，对梨形身材很友好，领口有手工制作的盘扣。 ]

示例2：针对一件羊绒大衣

    [ 我们这件大衣是100%阿拉善山羊绒，触感像云朵一样软糯。版型是经典的H型，不挑身材，长度刚好到小腿，走路带风。肩线做了落肩设计，可以弱化肩宽，看起来更柔和。纽扣用的是天然的牛角扣。 ]

示例3：针对一条阔腿裤

    [ 这条裤子用的是三醋酸面料，垂坠感特别好，而且抗皱免烫。裤腿是宽直筒设计，能修饰腿型，显得腿又长又直。腰部做了隐藏式松紧设计，穿着舒服不紧绷。]
```
