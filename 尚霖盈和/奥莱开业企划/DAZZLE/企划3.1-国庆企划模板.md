* **SWOT分析 (优势/劣势/机会/威胁):**

**业绩拆解**
* **核心公式:** 
	1. 业绩 = 成交客户数 × 客单价
	2. 客单价 = 平均吊牌价×折扣率×连带率
	3. 成交客户数 = 总客流量×进店率×试衣率×成交率
	4. 客流量 = 自然流量+邀约流量
- **业绩解构：**
	- 成交客户数 = 业绩 / 客单价 = 2577000 / 2700 = 955 人
	- 客流量 = 成交客户数 / （进店率 x 试衣率 x 试衣成交率）=955 / （10% x ）
