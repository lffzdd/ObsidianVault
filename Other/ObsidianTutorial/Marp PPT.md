### 4. 预览幻灯片

  

Marp for VS Code 扩展允许你在 VS Code 中直接预览你的幻灯片。

  

1. 打开 Markdown 文件。

2. 点击右上角的预览按钮，或按 `Ctrl+K V`（Windows/Linux）或 `Cmd+K V`（macOS）打开预览窗口。

3. 你将看到以幻灯片形式呈现的 Markdown 文件内容。

  

---

  

### 5. 导出为 PDF 或 HTML

  

你可以将你的幻灯片导出为 PDF 或 HTML 格式。

  

1. 打开 Markdown 文件。

2. 点击右上角的三个点图标，然后选择 “Marp: Export Slide Deck”。

3. 选择导出格式（PDF 或 HTML）。
